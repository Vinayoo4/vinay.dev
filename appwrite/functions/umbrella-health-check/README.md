# Umbrella Health Check Function

This is a simple Appwrite function to verify that the functions deployment pipeline and execution environment are set up correctly.

## Purpose

- Confirms your Appwrite Functions pipeline is working.
- Checks runtime and logging functionality without requiring secrets or complex setup.

## Deployment

Deploy this function manually through the Appwrite Console or using the Appwrite CLI.

```bash
appwrite init functions
appwrite push functions
```
